### 3.4 音效编程模式

```javascript
// Unity/C# 音效管理器
class AudioManager {
    // 对象池: 预分配N个AudioSource
    // 优先级系统: BGM > 重要SFX > 环境SFX > UI音效
    // 随机化: 同动作3-5个变体随机选取(避免重复感)
    // 音高随机: pitch = 0.95~1.05 (微变自然)
    
    void PlaySFX(AudioClip[] clips, Vector3 position) {
        var clip = clips[Random.Range(0, clips.Length)];
        var source = pool.Get();
        source.pitch = Random.Range(0.95f, 1.05f);
        source.PlayOneShot(clip);
    }
}
```

## 六、PBR 材质速查

### 标准 PBR 贴图通道

```
BaseColor (sRGB):       固有色，无光照信息
Normal (Linear):        法线贴图，RGB=XYZ
Metallic (Linear):      金属度，0=非金属，1=金属
Roughness (Linear):     粗糙度，0=镜面，1=完全粗糙
AO (Linear):            环境光遮蔽
Height/Displacement:    高度/置换贴图
Emissive (sRGB):        自发光

金属材质: BaseColor=偏暗金属色, Metallic=1, Roughness=划痕程度
非金属:   BaseColor=真实颜色,   Metallic=0, Roughness=表面粗糙度
皮肤:     BaseColor+Subsurface(0.02-0.05), Roughness=0.4-0.6
木头:     Metallic=0, Roughness=0.5-0.8
```

### 智能材质参数(Substance格式)

```
常见材质参数范围:
├── 生锈铁: Metallic=1, Roughness=0.7-0.9, 加点颜色变化
├── 抛光钢: Metallic=1, Roughness=0.1-0.2
├── 陶瓷:   Metallic=0, Roughness=0.1-0.3, Subsurface=0.01
├── 塑料:   Metallic=0, Roughness=0.3-0.6
├── 布料:   Metallic=0, Roughness=0.8-1.0, 微Subsurface
├── 玻璃:   Metallic=0, Roughness=0.0-0.05, Transmission=1
├── 黄金:   Metallic=1, Roughness=0.2, BaseColor=FFD700
└── 皮革:   Metallic=0, Roughness=0.6-0.8, Subsurface=0.01
```
