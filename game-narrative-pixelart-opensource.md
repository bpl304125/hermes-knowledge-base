# 游戏叙事 + 像素艺术 + 开源项目

## 一、游戏叙事设计

### 叙事结构
```
三幕剧: 建立→冲突→解决
英雄之旅: 平凡世界→冒险召唤→拒绝→导师→跨越→考验→奖赏→回归
分支叙事: 对话树 + 条件分支 → 多结局
环境叙事: 场景+物品描述+碎片化文本 → 玩家拼凑故事
```

### 叙事工具
| 工具 | 用途 |
|------|------|
| Twine | 互动小说,免费 |
| Ink/Inkle | 分支对话引擎,Unity/Godot集成 |
| Yarn Spinner | 对话系统,开源 |
| Articy:Draft | 专业叙事设计 |
| Obsidian | 世界观管理 |
