## 任务分发规则

```
代码编写    → 🧠大脑(DeepSeek)
研究探索    → 🔬研究员(OpenRouter)
创意设计    → 🎨画师(千帆/百炼)
文档翻译    → 📝书记官(Kimi)
配音音效    → 🎬导演(MiniMax)
平台发布    → 📡通讯员(混元/MiMo)
批量处理    → 🔧工程师(硅基流动)
学术审核    → 🎯顾问(智谱)
```
