## 二、游戏类型推断

基于: 抖音游戏热榜 + 蓝色主题 + 休闲定位 + 144px图标

**最可能**: 飞行/跳跃/收集类休闲游戏

- 蓝色背景 = 天空场景
- 白色元素 = 云朵/文字/高光
- 深色元素 = 角色/障碍物
- 小图标 + 抖音CDN = 抖音小游戏
