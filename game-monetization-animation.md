# 游戏变现 & 程序化动画

| 模式 | 收入 | 适合 | 实现难度 |
|------|------|------|---------|
| 激励视频(IAA) | $5-30 eCPM | 休闲/超休闲 | 低 |
| 插屏广告 | $3-10 eCPM | 关卡间 | 低 |
| Banner广告 | $0.5-2 eCPM | 底部 | 低 |

## 二、抖音广告设计

```
最佳广告位:
1. 复活 — 看广告继续游戏 (最高转化)
2. 奖励翻倍 — 看广告x2金币
3. 皮肤试用 — 看广告解锁临时皮肤
4. 每日加成 — 看广告领额外奖励

频率控制: ≤10次/天/用户
```

## 三、程序化动画

### IK 脚部放置
```
射线检测地面高度 → IK目标 = 命中点 + 偏移
锁脚(Planting): 落地后固定世界位置
骨盆补偿: min(左脚偏移, 右脚偏移)
```

### 注视系统
```
优先级: 眼睛(球体约束) > 头部(球体约束) > 脊柱(度数约束)
目标权重: 玩家 > 威胁 > 兴趣点 > 随机
```

### 物理动画混合
```
轻推: 动画90% + 物理10%
重击: 动画10% + 物理90%  
死亡: 纯物理(布娃娃)
弹簧力: F = k*(动画姿态 - 当前姿态)
```
