## 一、核心玩法 (Brain拆解)

### 玩法循环
```
规划道路 → 划区建造 → 水电供应 → 市民入住 → 纳税收钱 → 解锁新建筑 → 扩大城市
     ↑                                                              ↓
     └──────────── 港口贸易 ← 工厂生产 ← 资源采集 ←────────────────┘
```

### 三大支柱
| 支柱 | 系统 | 爽点 |
|------|------|------|
| 🏗️ **建造** | 道路/住宅/工业/商业 | 从空地到繁华都市的成就感 |
| 📊 **经营** | 电力/水源/治安/教育/医疗 | 平衡收支的精算快感 |
| 🚢 **贸易** | 港口货运/玩家交易/地标解锁 | 全球化扩张的史诗感 |

## 二、建筑体系

### 建筑分类
```
住宅区: 木屋→公寓→别墅→摩天楼 (人口)
商业区: 杂货铺→商场→CBD (税收)
工业区: 作坊→工厂→高新园区 (资源)
基础设施: 水塔/电站/警局/医院/学校 (满意度)
地标: 应天门/大雁塔/东方明珠 (吸引力)
交通: 土路→石板路→高速→地铁 (效率)
```

## 三、经营数值

| 资源 | 产生 | 消耗 | 短缺后果 |
|------|------|------|---------|
| ⚡电力 | 风电/火电/核电站 | 所有建筑 | 停工/人口流失 |
| 💧水源 | 水塔/水库 | 住宅/工业 | 疾病/火灾 |
| 🛡️治安 | 警局/巡逻 | 人口密度 | 犯罪/暴动 |
| 📚教育 | 学校/大学 | 升级需求 | 无法解锁高级建筑 |
| 🏥医疗 | 诊所/医院 | 污染/灾害 | 死亡率上升 |

## 四、灾难系统

| 灾难 | 触发 | 效果 | 重建奖励 |
|------|------|------|---------|
| ☄️陨石雨 | V博士事件 | 随机摧毁3×3区域 | 双倍重建速度 |
| 🌊洪水 | 暴雨+排水不足 | 低洼区淹没 | 解锁堤坝 |
| 🔥火灾 | 工业区过热 | 建筑损毁 | 消防局升级 |
| 😷瘟疫 | 医疗不足 | 人口骤降 | 医院免费扩容 |
| 🌪️地震 | 随机 | 大面积损毁 | 全部资源返还50% |

## 五、贸易系统

```
港口等级 → 航线数量 → 贸易伙伴 → 稀缺资源
  1级        1条        邻城        木材/石料
  2级        2条        州际        丝绸/瓷器
  3级        3条        海外        香料/科技
  4级        5条        全球        地标碎片

玩家交易: 市场建筑 → 挂单/求购 → 好友城市直连
```
